Os cmdlets de serviços e função de servidor que vêm com o Microsoft Lync Server 2013 permitem que você retorne informações sobre os vários componentes que foram instalados como parte de sua implantação do Lync Server 2013. Além disso, esses cmdlets facilitam a interrupção e o início dos serviços do Lync Server.
