_**Tópico da última modificação:** 2012-06-27_

Os cmdlets de interoperabilidade são usados para configurar a autenticação de servidor para servidor e a autorização entre o Microsoft Lync Server 2013 e outros produtos de servidor como o Microsoft Exchange Server 2013. A autenticação de servidor para servidor e autorização permite que esses servidores troquem e compartilhem dados perfeitamente.
