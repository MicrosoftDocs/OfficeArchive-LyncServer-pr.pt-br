# <a name="monitoring-starting-and-stopping-the-persistent-chat-services-in-lync-server-2013"></a>Monitorando, iniciando e interrompendo serviços do chat persistente no Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Tópico da última modificação:** 2012-10-06_

Os serviços de chat persistente e os serviços de conformidade de chat persistente fazem parte da topologia do Lync Server 2013 e, portanto, podem ser monitorados, interrompidos e iniciados usando os cmdlets do Windows PowerShell, **Get-CsWindowsService**, **Stop-CsWindowsService** e **Start-CsWindowsService**, respectivamente.
